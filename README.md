This projects implements a simple CRUD operations in a book store.
User can see the list of available books which are being fetchd from a local json file and serverd locally by json server to provide endpoints.
You can rate the book which triggers metho PATCH to update the value in the json file.
You can also Add to favorites which will trigger methos POST to create a new item in the favorites section in the json file,
You can also remove a book from the favorites section which will trigger DELETE method to delete from the json file and remove from ui.

Overally it is a single Page app for books
